# 🏜️ Offroad Semantic Scene Segmentation for Autonomous Vehicles
https://huggingface.co/rohan9977/segformer-b4-offroad-segmentation


## 📌 Project Overview
This project tackles the critical challenge of off-road perception for Unmanned Ground Vehicles (UGVs). Unlike structured city driving, off-road environments lack lane lines and predictable rules. To navigate safely, autonomous systems must understand the environment at a pixel level.

This repository contains a state-of-the-art semantic segmentation pipeline trained entirely on synthetic desert environments generated by **Duality AI's Falcon digital twin platform**. By bridging the sim-to-real gap, this model enables UGVs to accurately identify terrain, avoid obstacles, and plan safe paths.

## 🧠 Model Architecture & Details
To capture both high-resolution localized details (like small rocks) and broad spatial context (like the sweeping desert horizon), this project leverages a Transformer-based architecture rather than a traditional CNN.

* **Architecture:** SegFormer-B4
* **Backbone:** MiT-B4 (Mix Transformer encoder)
* **Model Size:** 64 Million Parameters
* **Pre-training:** ADE20K (150 classes)
* **Fine-tuning Dataset:** Duality AI Falcon synthetic desert images
* **Input Resolution:** 512 × 512 pixels
* **Output:** 10 Distinct Semantic Classes

## 🛠️ Environment Setup & Hardware

**System Requirements**
The inference scripts and data pipelines have been successfully tested locally on machines with a 13th Gen Intel Core i5-13420H and 16.0 GB of RAM. However, for full model training and hyperparameter tuning, cloud-based GPUs (e.g., NVIDIA T4, A100) are highly recommended.

**Installation**
We recommend using a clean virtual environment or `conda` via `conda-forge`.
